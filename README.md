## Why Copy-Paste?

Unlike traditional UI libraries that install as packages, CraftReactNative Templates uses a **copy-paste approach**:

- **Zero dependency risk** - No version conflicts or breaking changes
- **Full code ownership** - Modify anything without restrictions
- **No black boxes** - See exactly what your screens do
- **Production-ready** - Battle-tested templates you can trust
- **Perfect for teams** - No package updates breaking your app

## Overview

Production-ready screen templates covering common mobile app patterns — onboarding, messaging, trading, editorial, settings, and more. Each template includes dark mode support, animations, and follows React Native best practices.

Built with:

- [React Native Unistyles](https://github.com/jpudysz/react-native-unistyles) for theming
- [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/) for animations
- [CraftReactNative UI](https://github.com/craftreactnative/ui) components
- TypeScript with strict mode

Templates are compatible with any React Native project — Expo or bare workflow. The included demo app uses Expo for convenience.
